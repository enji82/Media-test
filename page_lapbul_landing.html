Pilih Jenjang Sekolah

Silakan pilih jenjang laporan bulan yang ingin Anda unggah.

Laporan PAUD

Input data siswa, PTK, dan unggah dokumen untuk TK/KB/SPS.

Laporan SD

Input data rombel, murid, guru, dan unggah dokumen untuk Sekolah Dasar.

  • Laporan bulan dilaporkan dengan cara mengunggah scan PDF secara mandiri.
  • Data digunakan sebagai laporan keadaan sekolah setiap bulannya ke Korwil Disdikbud.
  • File PDF yang diunggah adalah scan laporan bulan ASLI yang sudah ditandatangani.
  • Ukuran file maksimal adalah 300 KB.
  • Laporan wajib diunggah sebelum tanggal 10 bulan berikutnya.
  • Disarankan untuk tidak memindai menggunakan HP, gunakan mesin scanner untuk hasil terbaik.
